# Лабораторная работа №1. Быстрое возведение в квадрат

Метод быстрого возведения в квадрат реализован как дополнительный метод в программе Big_Number (лабораторная работа по ссылке: https://github.com/irishN24/Class_Big_Number/blob/main/B_N.cpp)

## Описание метода `fast_square` класса `Big_Number`

Метод `fast_square` реализует **быстрый алгоритм возведения в квадрат** больших целых чисел, представленных в системе счисления с основанием `BASE_SIZE` (размер типа `Base` в битах). Алгоритм основан на оптимизации стандартного умножения, учитывая симметрию при возведении в квадрат.

## Принцип работы

### Математическая основа
При стандартном умножении двух чисел A и B длины n требуется n² операций умножения. При возведении в квадрат (A·A) можно использовать свойство:
- Произведения вида aᵢ·aⱼ и aⱼ·aᵢ равны, поэтому их можно вычислять один раз и удваивать.

### Пошаговый алгоритм

1. 
   - Создаётся число-результат `res` с максимальной длиной `2*len + 1`
   - Все коэффициенты результата обнуляются

2. 
   Для каждого индекса i от 0 до len-1:

   **2.1 Вычисление квадрата диагонального элемента**
   ```
   t = res.coef[2*i] + coef[i] * coef[i] // вычисляются две цифры u и v
   res.coef[2*i] = младшая часть t, т.е v
   carry = старшая часть t (перенос)
   ```

   **2.2 Вычисление удвоенных перекрёстных произведений**
   Для каждого j от i+1 до len-1:
   ```
   add2 = 2 * coef[i] * coef[j] // 2*x_i*x_j
   t2 = res.coef[i+j] + add2 + carry
   res.coef[i+j] = младшая часть t2
   carry = старшая часть t2
   ```

   **2.3 Обработка оставшегося переноса**
   Пока перенос не равен нулю:
   ```
   t3 = res.coef[i+len] + carry
   res.coef[i+len] = младшая часть t3
   carry = старшая часть t3
   i+len++
   ```

3. **Корректировка длины**


## Преимущества алгоритма
- **Сокращение количества операций**: примерно в 2 раза меньше умножений по сравнению со стандартным умножением
- **Эффективность**: особенно заметна на больших числах (например, 1000+ разрядов)

## Используемые типы данных
- `Base` — тип для одного разряда
- `W_Base` — расширенный тип для промежуточных вычислений (предотвращает переполнение)
- Перенос (carry) сохраняется между итерациями

# Лабораторная работа №2: Дихотомический алгоритм возведения в степень

Программа реализует класс `Big_Number` для работы с длинными целыми числами произвольной длины и демонстрирует эффективный алгоритм возведения в степень с использованием метода **быстрого возведения в степень (бинарного/дихотомического алгоритма)**.

## Основная идея

Метод **бинарного возведения в степень** (также известный как "exponentiation by squaring" или "дихотомический алгоритм") реализован как дополнительный метод в программе Big_Number (лабораторная работа по ссылке: https://github.com/irishN24/Class_Big_Number/blob/main/B_N.cpp). Этот подход позволяет вычислить \( a^n \) за \( O(\log n) \) операций умножения вместо \( O(n) \) операций при наивном подходе.

## Принцип работы

### Математическая основа 

1. Двоичное представление показателя степени. Показатель степени n преобразуется в двоичную систему счисления.
2. Свойство возведения в квадрат. Для любого числа a и чётного n выполняется тождество:  `a^n = (a^(n/2))^2 = a^(n/2) · a^(n/2)`. Это позволяет свести задачу к вычислению степени с вдвое меньшим показателем, потратив только одну операцию умножения. 
3. Преобразование для нечётного показателя. Если n нечётно, то: `a^n = a^(n-1) · a`. Таким образом, нечётная степень преобразуется в чётную, что упрощает дальнейшие вычисления.

### Пошаговый алгоритм

1. **Определение длины показателя в битах**
   - Вычисляется количество значащих битов в показателе степени через метод `bitLength()`

2. **Обработка особых случаев**
   - \( a^0 = 1 \) — любое число в нулевой степени даёт 1
   - \( a^1 = a \) — число в первой степени равно самому себе
   - \( 1^n = 1 \) — единица в любой степени остаётся единицей
   - \( 0^0 = 1 \) — математическое соглашение
   - \( 0^n = 0 \) (при \( n > 0 \)) — ноль в положительной степени равен нулю

3. **Основной цикл по битам показателя**
   ```
   result = 1
   base = исходное число
   
   для каждого бита показателя от младшего к старшему:
       если текущий бит == 1:
           result = result * base
       base = base²  (возведение в квадрат)
   ```

4. **Использование быстрого возведения в квадрат**
   - Метод `fast_square()` реализует эффективный алгоритм умножения числа на себя
   - Сложность возведения в квадрат оптимизирована по сравнению с обычным умножением

Программа включает тестирование, которое сравнивает время выполнения обоих методов и проверяет корректность результатов.
